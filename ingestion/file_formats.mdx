## Supported file formats

Ingestor can process the following file formats:

- **email**
- **pdf**
- **docx**
- **doc**
- **xlsx**
- **xls**
- **csv**
- **xml**
- **jpg**
- **jpeg**
- **png**
- **gif**
- **bmp**
- **tiff**
- **pptx**
- **ppt**

<Note>
If you need support for a file type not listed here, please [contact us](mailto:support@ingestor.io).
</Note>
