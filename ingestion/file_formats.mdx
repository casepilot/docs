## Supported file formats

Ingestor can process the following file formats:

- **email**
- **pdf**
- **docx**
- **doc**
- **xlsx**
- **xls**
- **csv**
- **xml**
- **jpg**
- **jpeg**
- **png**
- **gif**
- **bmp**
- **tiff**
- **pptx**
- **ppt**
